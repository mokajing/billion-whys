三、信息使用范围

3.1 允许的使用方式

  1. 内容展示：将素材库中的"为什么"问题、答案、插图呈现给用户；
  2. 基础运行：依据屏幕尺寸做 UI 适配；
  3. 时间管理：依据本地 storage 中的累计时长触发休息提示；
  4. 举报处理：在监护人主动举报后，将举报内容快照用于人工审核。

3.2 严禁的使用方式

本小程序明确禁止以下行为：

  1. ❌ 不进行用户画像（profiling）；
  2. ❌ 不使用任何推荐算法（recommendation algorithm）；
  3. ❌ 不基于使用行为做个性化推送；
  4. ❌ 不进行 A/B 测试与行为分析埋点；
  5. ❌ 不将使用数据用于商业变现、广告竞价、数据交易；
  6. ❌ 不进行精准营销或定向投放；
  7. ❌ 不向未成年人推送任何商业信息。

3.3 自动化决策声明

本小程序不使用任何自动化决策机制对用户产生法律效力或类似重大影响，不进行自动化画像、自动化推荐、自动化定价。

六、未成年人特别保护条款

6.1 适用对象

本条款适用于 2-6 岁儿童及其监护人。本小程序假设所有使用者均为未成年人，采取最高保护标准。

6.2 监护人同意机制

  1. 首次启动小程序时，强制进入"家长验证页"（详见《未成年人保护方案》第 1 章）；
  2. 监护人完成简单问答或扫码验证后，方可进入主界面；
  3. 进入"成人科普"页面时，二次要求监护人确认；
  4. 未通过验证的，一律按未成年人模式呈现内容。

6.3 内容分级承诺

  1. 2-6 岁主界面内容严格遵守"三无"标准：无暴力、无恐怖、无性暗示；
  2. 涉及"流血 / 做梦 / 死亡"等敏感话题，仅以"成人科普"层隐藏呈现，需家长验证后查看；
  3. 实验类内容强制提示"家长陪同"。

6.4 时间管理承诺

  1. 单次使用 ≤ 15 分钟，每日累计 ≤ 30 分钟；
  2. 22:00-次日 6:00 默认禁用（监护人可在控制面板关闭）；
  3. 时间到达后强制退出至家长验证页。

6.5 商业化限制

  1. 未成年人内容无任何广告；
  2. 无虚拟支付、无内购；
  3. 无诱导分享 / 关注 / 转发；
  4. 无排行榜 / 无竞技机制。

6.6 不收集承诺

本小程序郑重承诺：不以任何形式收集、存储、使用未成年人个人信息。所有使用数据均存储于本地，监护人可随时清除。

7.3 删除权

监护人可通过以下方式删除数据：

  1. 一键清除本地数据：家长控制面板 > 重置隐私协议同意；

  3. 卸载小程序：卸载后微信将自动清除本地 storage。

7.4 撤回同意权

监护人可随时通过"家长控制面板 > 撤回同意"撤回对本政策的同意，撤回后小程序将无法继续使用，直至监护人重新同意。

7.5 账号注销

本小程序不要求用户注册账号，因此不涉及账号注销。本地数据清除等同于"注销"。

7.6 响应时限

我们将在收到监护人请求后 15 个工作日内予以响应，复杂情况不超过 30 个工作日。
