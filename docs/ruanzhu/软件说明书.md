## 一、健康使用忠告

适度使用有益成长，沉迷屏幕伤眼伤身。请家长陪同使用，合理安排时间，享受亲子共学的乐趣。本软件适合 2-6 岁儿童在家长陪同下使用，单次使用时长建议不超过 15 分钟。

## 三、全流程图文介绍

### 3.1 启动与主界面

软件启动后，首次使用需家长验证（回答 3 道简单问题），通过后进入「发现」主页面。

**发现页**：展示每日推荐问题、热门问题分类入口（身体/动物/食物/家/自然/社会），问问兔和答答熊 IP 形象欢迎用户。

### 3.2 核心功能流程

#### 流程一：浏览学习

```
发现页 → 选择分类 → 浏览问题列表 → 点击问题 → 查看三层答案 → 展开家长科学原理 → 查看亲子实验
```

#### 流程二：提问

```
问一问页 → 输入问题 → 系统匹配 → 显示答案 → 相关问题推荐
```

#### 流程三：个人中心

```
我的页 → 查看学习进度 → 家长控制面板 → 使用时长统计 → 举报入口
```

### 3.3 结束/退出

用户可随时通过微信返回退出小程序，学习进度自动保存。

## 五、可体验的全部系统

### 5.1 内容系统

- 300+ 个"为什么"问答
- 6 大分类：身体（body）、动物（animals）、食物（food）、家（home）、自然（nature）、社会（society）
- 每题包含：三层答案 + 科学原理 + 亲子实验 + IP 场景 + 配图

### 5.2 三层答案系统

| 层级 | 名称 | 适用年龄 | 说明 |
|---|---|---|---|
| Layer 1 | 直觉比喻 | 2-3 岁 | 用比喻直观解释 |
| Layer 2 | 机制解释 | 4-5 岁 | 讲述原理机制 |
| Layer 3 | 完整科普 | 5-6 岁 | 完整科学解释 |

### 5.3 家长科学专区

- 准确的科学原理说明
- 仅家长验证后可见
- 不替代专业医疗建议

### 5.4 亲子实验系统

- 每题配一个可在家完成的实验
- 含材料清单、操作步骤、安全提示
- 需家长全程陪同

### 5.5 未成年人保护系统

- 首次启动家长验证
- 单次使用 ≤ 15 分钟
- 每日累计 ≤ 30 分钟
- 内容分级（layer1/2/3 + science + experiment）
- 举报不当内容入口
- 不收集未成年人个人信息

### 5.6 图片资源系统

- 每题 5 张插画（layer1/2/3 + science + experiment）
- 儿童绘本风格，暖色水彩
- 通过 CDN 加载，384px WebP 格式

### 5.7 数据持久化系统

- 学习进度本地存储
- 家长验证状态
- 使用时长统计
- 收藏记录
