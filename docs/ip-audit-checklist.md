# IP 审计 Checklist

> 创建日期：2026-07-07 V8.74 Sprint 73 第138轮
> 负责人：墨小暖
> 用途：每次内容发布前检查问问兔和答答熊 IP 形象一致性

---

## 问问兔 (Ask Bunny) 形象检查

### 视觉一致性
- [ ] 耳朵角度：好奇时前倾 12°（非 15°，非竖直）
- [ ] 耳朵颜色：外白内粉（#F5B7C8）
- [ ] 身体比例：头身比约 2:1（Q版圆润）
- [ ] 毛色：纯白 (#FFFFFF)
- [ ] 眼睛：黑色圆点 (#2B2B2B)，尺寸 1.8px
- [ ] 鼻子：粉色三角形 (#E89AAB)
- [ ] 整体色调：温暖明亮，无冷色调

### 性格一致性
- [ ] 好奇心驱动：每个场景中问问兔在"提问"或"探索"
- [ ] 表情：睁大眼睛、耳朵前倾（好奇姿态）
- [ ] 动作：不出现攻击性动作或负面表情
- [ ] 语言：用"为什么"、"咦"、"哇"等好奇语气词

### 场景规则
- [ ] 科学实验场景：问问兔在观察/提问，答答熊在解释/操作
- [ ] 日常场景：问问兔在发现/提问
- [ ] 情绪安抚场景：问问兔可以有困惑表情，但不能有恐惧表情

---

## 答答熊 (Buddy Bear) 形象检查

### 视觉一致性
- [ ] 体型：比问问兔大一圈（约 1.3 倍），给人安全感
- [ ] 毛色：暖棕色系
- [ ] 表情：温和微笑，可靠稳重
- [ ] 动作：引导/保护姿态，不出现支配性动作
- [ ] 整体色调：温暖安心

### 性格一致性
- [ ] 知识引导者：每个场景中答答熊在"解释"或"引导"
- [ ] 语气：温和、"我们一起来看看"、"别担心"
- [ ] 不出现：训斥表情、不耐烦姿态、居高临下感

### 场景规则
- [ ] 科学实验场景：答答熊在演示/说明
- [ ] 安全提示场景：答答熊在守护/提醒
- [ ] 不能出现：答答熊"不知道"或"不会"的姿态（与"知识引导者"人设冲突）

---

## 双人互动规则

- [ ] 身高差：问问兔约到答答熊肩膀
- [ ] 互动距离：亲近但不挤压（保持 1-2 个问问兔身位）
- [ ] 眼神交流：问问兔看答答熊（期待回答），答答熊看问问兔或看实验（引导）
- [ ] 不能出现：答答熊不理问问兔、问问兔独自害怕

---

## AI 插画 Prompt 审查要点

- [ ] 耳朵角度是否正确（12° 前倾）
- [ ] 颜色是否在 IP 色板范围内
- [ ] 双人互动是否符合规则
- [ ] 表情是否符合场景情绪（好奇/温和/安心）
- [ ] 背景是否为暖色调（禁用冷灰/蓝色调背景）
