## Data We DO NOT Collect

| Category | Collected? | Notes |
|----------|-----------|-------|
| Child's real name | No | Not required |
| Child's date of birth | No | Only age range (2-6) is used for content adaptation |
| Geolocation | No | No GPS or location tracking |
| Photos or videos | No | No camera or media access |
| Contacts | No | No address book access |
| Device ID | No | No device fingerprinting |
| Biometric data | No | No face/voice recognition |
| Health data | No | No health-related data collection |

## Data We DO Collect (Minimal, Device-Only)

| Category | Purpose | Storage |
|----------|---------|---------|
| Age range (2-3, 3-4, 4-5, 5-6) | Content age-appropriate matching | Local device only |
| Age preference ("小宝"/"哥哥姐姐") | UI layer adaptation per user-selected age group | Local device only, not transmitted to server |
| Age preference (bw_age_preference) | Persists user's age group selection (3-4/4-5/5-6) across sessions for seamless content adaptation | localStorage only, never uploaded |
| Language preference (bw_locale) | UI language display (zh/en) | localStorage only, never uploaded |
| Browsing history (bw_view_history) | "Continue exploring" feature — shows recently viewed questions | localStorage only, anonymous question IDs, never uploaded |
| Favorite questions (bw_favorites) | Quick access to saved content | localStorage only, anonymous question IDs, never uploaded |
| Answer feedback (emoji-based) | Content quality improvement | Local device only |

## Key Principles

1. **Zero Child Data Collection**: We do not collect, store, or transmit any personal information about children under 14.
2. **Device-Only Storage**: All user preferences (favorites, feedback, age range, view history) are stored exclusively on the user's device. No cloud sync.
3. **No Third-Party Analytics**: We do not use any third-party analytics SDKs or tracking services.
4. **No Advertising**: This app contains no advertisements and no ad-related data collection.
5. **Parental Consent**: The app includes a parent verification gate before any interactive features are accessible to children.

## Parental Rights
